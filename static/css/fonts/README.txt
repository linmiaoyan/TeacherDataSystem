请将Bootstrap Icons字体文件放在此目录下：

需要下载以下两个文件：
1. bootstrap-icons.woff2
2. bootstrap-icons.woff

下载方式：
1. 访问 https://github.com/twbs/icons/releases
2. 下载最新版本的 bootstrap-icons-x.x.x.zip
3. 解压后，将 fonts 文件夹中的 bootstrap-icons.woff 和 bootstrap-icons.woff2 复制到此目录

或者直接下载字体文件：
- https://github.com/twbs/icons/raw/main/font/fonts/bootstrap-icons.woff2
- https://github.com/twbs/icons/raw/main/font/fonts/bootstrap-icons.woff

如果网络不好，可以：
1. 使用手机热点下载
2. 或者暂时使用CDN（修改 bootstrap-icons.css 中的路径为CDN地址）
